Pseudoclasses: âncoras

  • Servem para alterar a formatação dos links <a href>
  • Existem quatro estados para um link:
    • a:link é o link em estado de "repouso"
    • a:visited é o link depois de ser visitado
    • a:hover é o link quando o cursor do mouse passa sobre ele
    • a:active é o link sendo clicado
a:link { color:red; } a:visited { color:black; } a:hover { color: white; background-color: red; } a:active { color:green; background-color: white; }
